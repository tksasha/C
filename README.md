## How to build and run `src/main.c`:
```
make
bin/main
```

## How to build and run `src/ex/*.c`:
```
make src/ex/FILE.c
bin/main
```

## How to build and run exercises from K&R book:
```
make src/kre/EXERCISE.c
bin/main
```

To cleanup
```
make clean
```
